# RDCore.Diagnostics
**Extensilité | Extensibility**

**Rubberduck Core** se veut une plateforme de développement extensible à un niveau sans précédent pour VBA, et ceci débute avec l'implémentation des diagnostics en tant que plug-in (inclus). De cette façon, nous nous assurons d'offrir et de consommer le même API que celui qui servira à d'éventuelles extensions de tierces parties.


**Rubberduck Core** means to be a development platform for VBA extensible at an unprecedented level for VBA, and it begins with implementing the core diagnostics as a (bundled) plug-in. This way, we "dogfood" the extensibility API that can serve eventual third-party plug-ins.
