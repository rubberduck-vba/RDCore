**RDCore.SDK** implémente une interprétation des spécifications **MS-VBAL** qui se veut _corriger_ celle-ci, avec l'implémentation **MS-VBA** comme référence.
Il en résulte inévitablement ainsi **RD-VBA**, l'implémentation de cette même spécification définie dans ce SDK.

# RDCore SDK

## Welcome!

**RDCore.SDK** implements an interpretation of the **MS-VBAL** language specifications that means to _fix it_, using the **MS-VBA** implementation as a cross-reference.
The inevitable result is **RD-VBA**, the implementation of these specifications defined in this SDK.

> 👉 The **API** section of this site is **generated automatically** by compiling comments (xmldoc) in the source code:
this content is therefore unfortunately only available in English, to avoid creating unnecessary _noise_ in the source code.  
> 👉 The **Docs** section of this site contains documents (.md) integrated into the project repository and will ultimately contain **articles** and **tutorials** 
that _will_ be available in both French and English.
